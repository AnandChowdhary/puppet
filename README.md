## 💡 Usage

Install the package from [npm](https://www.npmjs.com/package/puppet):

```bash
npm install puppet
```

Import and use;

```ts
import { nodeTs } from "@anandchowdhary/node-ts";

nodeTs();
```

## 👩‍💻 Development

Build TypeScript:

```bash
npm run build
```

Run unit tests and view coverage:

```bash
npm run test-without-reporting
```
